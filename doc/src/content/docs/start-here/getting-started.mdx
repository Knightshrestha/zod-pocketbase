This library adds Zod safety with ease to your PocketBase instance.

## Installation

To install `zod-pocketbase`, run the following from your project directory:

<Tabs>
  <TabItem label="npm">
    ```shell
    npm install zod-pocketbase
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```shell
    pnpm add zod-pocketbase
    ```
  </TabItem>
  <TabItem label="yarn">
    ```shell
    yarn install zod-pocketbase
    ```
  </TabItem>
</Tabs>
